Ads Master for wordpress to display ads in your website at your custom setting.

== Description ==

Ads Master allows you to display ads inside your website in any template, widget, position or even inside pages and posts.

Its mainly developed for blogger to display various type of ads in blogging website.

== Installation ==

1. Upload the `ads-master` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enter your Google Adsence Code by clicking the Ads Master under Settings menu.
4. Select and manage your setting to manage ads.


== Frequently Asked Questions ==

= Can I create a dynamic responsive add ? =

Yes, if your program offers responsible add, use this format, and define the maximum heigh in the pluging settings. Then the width will automatically adjust to the mobile device.

= Why can't I use it with AdSense ? =

Because it is against Google policy to place Google ads in a floating script that causes the ad to appear in a “sticky" position on the page as the user scrolls down".
